About SailfishOS.app

What is this?

SailfishOS.app is a community-driven compatibility database for Android apps on SailfishOS, similar to ProtonDB for Linux gaming or Wine AppDB.

How does it work?

We list popular Android apps and show their compatibility status on SailfishOS:

  • Native Apps: SailfishOS-native alternatives built specifically for the platform
  • Android App Support: How well the Android version works through SailfishOS's Android compatibility layer

Compatibility Ratings

Works App functions correctly
Partial Some features may not work
Does Not Work App is not usable
Unknown Not yet tested

Contributing

This database is crowdsourced! If you have tested an app on SailfishOS, your reports help other users know what to expect.

About SailfishOS

SailfishOS is a mobile operating system developed by Jolla. It's based on Linux and features a unique gesture-based user interface. SailfishOS can run Android apps through its Android App Support layer.
